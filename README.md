# Personal Task Manager

A full-stack **MERN** (MongoDB, Express, React, Node.js) application designed to help users organize their daily tasks. The project features secure user authentication, real-time task management, and a responsive interface.

## Live Demo

- **Backend API:** [https://personal-task-manager-j7gm.onrender.com](https://personal-task-manager-j7gm.onrender.com)
- **Frontend App:** [https://personal-task-manager-96oguwpy0-john-irishs-projects.vercel.app/](https://personal-task-manager-96oguwpy0-john-irishs-projects.vercel.app/)

---

## Features

- **User Authentication:** Secure Login and Registration using JSON Web Tokens (JWT).
- **Task Management:** Create, Read, Update, and Delete (CRUD) tasks.
- **Organization:** Categorize tasks by priority (Low, Medium, High) and set deadlines.
- **Responsive UI:** Built with React and CSS for a clean experience on desktop.
- **Secure Backend:** Protected routes ensuring users can only manage their own data.

---

## Tech Stack

- **Frontend:** React.js, Vite, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Atlas)
- **Authentication:** JWT (JSON Web Tokens), BCrypt
- **Deployment:** Render (Backend), Vercel (Frontend)
